## How to Use This Reference

### For Developers:
- Use the **English** column as the **key** (e.g., `data-i18n="shoulder"`)
- Each corresponding cell is the translation in that language

### For Testing:
- Switch to each language in the app
- Compare what you see with this reference table
- Make sure everything matches

### For Adding New Translations:
1. Choose a key name (use camelCase)
2. Add translations in all 4 languages
3. Add `data-i18n="keyName"` to HTML element
