Demo Exercise Videos
====================

Place MP4 demo video files in this directory.
They are served by the backend at:
  http://localhost:8001/static/demo_videos/<filename>

Expected filenames (must match EXERCISE_DEMO_MAP in script.js):

  Neck:
    neck_flexion.mp4
    neck_extension.mp4
    neck_rotation.mp4

  Shoulder:
    shoulder_flexion.mp4
    shoulder_extension.mp4
    shoulder_abduction.mp4
    shoulder_adduction.mp4
    shoulder_internal_rotation.mp4
    shoulder_external_rotation.mp4

  Elbow:
    elbow_flexion.mp4
    elbow_extension.mp4

  Wrist:
    wrist_flexion.mp4
    wrist_extension.mp4
    wrist_rotation.mp4

  Hip:
    hip_flexion.mp4
    hip_extension.mp4
    hip_abduction.mp4
    hip_adduction.mp4

  Knee:
    knee_flexion.mp4
    knee_extension.mp4

  Ankle:
    ankle_dorsiflexion.mp4
    ankle_plantarflexion.mp4
    ankle_inversion.mp4

  Squats / Back:
    squat.mp4
    wall_squat.mp4
    mini_squat.mp4
    sit_to_stand.mp4
    step_up.mp4
    back_extension.mp4

If a video file is missing for an exercise, the modal will show
a placeholder message with text-based posture tips instead.
